How It Works
Authentication Flow:
  1. User accesses https://hr.dgcf.ovh
  2. Cloudflare Access intercepts the request
  3. User is redirected to Authentik for login
  4. After successful auth, Cloudflare adds JWT headers
  5. Request reaches this application with identity info
  6. App reads headers and personalizes experience
Benefits:
  • No login form in the application
  • No password management in the app
  • Centralized authentication via Authentik
  • Additional policies (device, location, etc.)
  • Audit logs in Cloudflare dashboard
